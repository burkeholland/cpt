Get started

One install.
Every shell.

🍎

macOS

Homebrew or direct binary. Supports zsh, bash, and fish out of the box.

$ go install github.com/burkeholland/cpt@latest
🐧

Linux

Same binary, same experience. Works on X11 and Wayland, with clipboard support for both.

$ go install github.com/burkeholland/cpt@latest
⊞

Windows

Native PowerShell integration with PSReadLine. No WSL required.

> go install github.com/burkeholland/cpt@latest

Then bind it to Ctrl+K in your shell:

$ cpt --install
How it works

Think it.
Run it.

01

Describe, don't memorize

Type what you want in plain English. cpt calls GitHub Copilot to translate your intent into the right command for your shell — bash, zsh, fish, or PowerShell.

02

Review and refine

See the command before it runs. Not quite right? Type a follow-up to iterate — cpt regenerates instantly. When it's perfect, press enter and it drops right onto your prompt.

03

Zero footprint

cpt runs in your terminal's alternate screen buffer. When you're done, it vanishes completely — no leftover output, no clutter. Just the command on your prompt, ready to go.

04

Pick your model

Switch between any model in your Copilot subscription with tab. Claude, GPT, Gemini — whatever you prefer. cpt remembers your last choice.

05

Shell-native output

cpt knows which shell and OS you're running. On PowerShell you get PowerShell commands, on bash you get bash. No translating between ecosystems.
